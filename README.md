**OmniPulse** is a domain-agnostic Agentic MLOps pipeline designed specifically for high-noise transient detection. Previously known as the "Agentic-Wavelet Foundation Pipeline", OmniPulse combines the rigorous mathematics of the cascaded **Wavelet Scattering Transform (WST)** with a **TypeScript Agentic Orchestrator** to automatically parse, denoise, and analyze extreme time-series anomalies like EEG epileptic spikes or astrophysical Fast Radio Bursts (FRBs).

Created and maintained by **Samvardhan Singh**.

---

## What is OmniPulse?
Traditional signal processing pipelines are rigid and require constant hyperparameter tuning by domain experts. OmniPulse offloads this tuning and execution to an intelligent Agent. 

1. **Python Engine**: The `omnipulse` backend uses `kymatio` and `scikit-learn` to extract deep, translation-invariant frequency cascades from raw 1-D `.npy` waveforms. It exposes these mathematics as autonomous tools via the **Model Context Protocol (MCP)**.
2. **Agentic Orchestrator**: A Bun/TypeScript agent dynamically spawns the Python backend, ingests the data statistics, and makes autonomous decisions (Approve, Denoise, or Halt) using explicit statistical outliers (e.g., detecting `NaN`/`Inf` corruption or extreme variance shifts).

## Installation

The core mathematical engine is available on PyPI:

```bash
pip install omnipulse
```

*(Note: Requires Python 3.10+)*

To run the full pipeline (including the dual-agent TypeScript orchestrator):

```bash
git clone https://github.com/samvardhan03/OmniPulse.git
cd OmniPulse
pip install -e "./open-source-wst"
cd agent
bun install
```

## Quick Start (E2E Dry Run)

You can test the entire autonomous pipeline by creating a batch of synthetic corrupted signals and letting the TS orchestrator manage it:

```bash
# 1. Generate synthetic data (50 signals; 10 transients, 2 anomalies)
python open-source-wst/scripts/generate_sim_data.py

# 2. Run the Agentic Orchestrator
cd agent
bun run src/main.ts process --input ../data/raw_sim --output ../data/processed
```

The agent will automatically execute the Wavelet transform, dimensionally reduce via PCA, catch the extreme anomalies generated by the simulation, and print a clear `DATA_CORRUPTION_HALT` verdict.
